# 项目简介 python571

本项目是一个集游戏数据推荐系统、数据爬取以及可视化功能于一体的综合性平台。以下为该项目的详细说明。

## 技术栈

- **后端**: Python
- **框架**: Flask
- **数据库**: MySQL
- **可视化**: ECharts
- **推荐算法**: 协同过滤算法

## 功能模块

- 用户登录注册
- 数据展示和分析
- 游戏搜索与推荐

## 系统角色

- 用户: 可登录注册，进行游戏搜索与推荐功能的使用。
- 系统管理员: 负责系统维护和数据分析。

## 运行环境

- Python环境
- Flask框架支持
- MySQL数据库环境
- 适用于主流浏览器

## 部署步骤

1. 环境搭建：配置Python环境，安装Flask、MySQL及所需依赖。
2. 数据爬取：使用爬虫程序从Steam网站抓取游戏数据。
3. 数据处理：清洗和格式化爬取的数据，并存储至MySQL数据库。
4. 系统开发：利用Flask框架搭建Web应用，集成ECharts进行数据可视化。
5. 功能测试：进行用户功能测试，确保系统稳定性。

## 目录结构

```
project/
│
├── app/              # Flask应用目录
│   ├── templates/    # HTML模板目录
│   └── static/       # 静态资源目录
│
├── database/         # 数据库相关
│
├── src/              # 源代码目录
│   ├──爬虫/           # 爬虫相关代码
│   └──推荐系统/        # 推荐系统相关代码
│
└── venv/             # 虚拟环境目录
```

## 核心亮点

- **高度集成的数据爬取**: 项目内置爬虫，能够从指定网站抓取游戏数据，并进行有效的数据清洗和存储。
- **协同过滤推荐算法**: 利用先进的协同过滤算法，为用户提供个性化游戏推荐。
- **丰富的数据可视化**: 通过ECharts实现数据的直观展示，助力用户洞察游戏数据趋势。
